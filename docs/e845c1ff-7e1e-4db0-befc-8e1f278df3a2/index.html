Proxmox Cluster Installation

1. Setup Proxmox Virtual Environment(PVE)​

  • Boot into ISO Proxmox-7.4 and Install

  • OS DISK: ZFS RAID-1 (2 X SSD) - For Proxmox OS
  • STORAGE DISK: ZFS RAID-1 (2 X HDD) - For VM Server

  • Set time zone

  • Set password

  • Setup Network

  • Summary information

  • Installation in progress

  • The server will be rebooted

  • When the installation is finished, log in to PVE via WEB-INTERFACE https://ip-promox:8006 to Proxmox Standalone Mode

From now, Setting-up everything can be done via Web-Interface, do the same for another server.

  • Update, and upgrade software

  • PVE dashboard

2. Configure A Cluster​

  • Create Cluster on the first PVE-01

  • Copy Cluster information to join another server

  • From PVE-02

  • Do the same for PVE-03

  • After joining Cluster, PVE-02, and PVE-03 temporarily lost connection because they were restarting services.
  • Now we can manage them via a single Dashboard
